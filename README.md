# Portfólio Online - Nilson Miranda

Site criado para a atividade prática de Fundamentos da Programação Web.

## Requisitos atendidos

- HTML5, CSS3 e JavaScript puro, sem frameworks.
- Navegação por âncoras em uma única página.
- Seções obrigatórias: Sobre mim, Formação, Portfólio e Contato.
- Menu visível e responsivo.
- Alternância entre tema claro e escuro com JavaScript.
- Formulário de contato com validação de nome, e-mail e mensagem.
- Simulação de envio com mensagem de sucesso.

## Arquivos principais

- `index.html`: estrutura e conteúdo do portfólio.
- `styles.css`: diagramação, responsividade e temas.
- `script.js`: menu responsivo, tema, link ativo e validação do formulário.
- `entrega.md`: modelo com campos para preparar o PDF final da atividade.

## Publicação no GitHub Pages

1. Crie um repositório público no GitHub, por exemplo `portfolio-nilson-miranda`.
2. Envie estes arquivos para a branch principal do repositório.
3. No GitHub, acesse `Settings > Pages`.
4. Em `Build and deployment`, selecione a branch principal e a pasta `/root`.
5. Aguarde a publicação e copie o link gerado.

O link esperado ficará parecido com:

```text
https://nilsonpmjr.github.io/portfolio-nilson-miranda/
```
